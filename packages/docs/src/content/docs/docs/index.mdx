## Core Features

<CardGrid>
  <Card title="Workflow Execution" icon="setting">
    Define multi-step processes with node graphs. Each node contains a directive (what to do) and
    completion condition (when done).
  </Card>
  <Card title="MCP Protocol" icon="puzzle">
    Standard Model Context Protocol integration. Works with Claude Code, Cursor, and any
    MCP-compatible client.
  </Card>
  <Card title="Template Engine" icon="document">
    Dynamic content with variable interpolation. Access execution context, results, and metadata in
    your workflows.
  </Card>
  <Card title="Multi-Language" icon="translate">
    Full i18n support. Create workflows and documentation in multiple languages.
  </Card>
</CardGrid>
