## Основные возможности

<CardGrid>
  <Card title="Выполнение воркфлоу" icon="setting">
    Определяйте многошаговые процессы с помощью графов узлов. Каждый узел содержит директиву (что
    делать) и условие завершения (когда готово).
  </Card>
  <Card title="Протокол MCP" icon="puzzle">
    Стандартная интеграция Model Context Protocol. Работает с Claude Code, Cursor и любым
    MCP-совместимым клиентом.
  </Card>
  <Card title="Шаблонный движок" icon="document">
    Динамический контент с подстановкой переменных. Доступ к контексту выполнения, результатам и
    метаданным.
  </Card>
  <Card title="Мультиязычность" icon="translate">
    Полная поддержка i18n. Создавайте воркфлоу и документацию на разных языках.
  </Card>
</CardGrid>
